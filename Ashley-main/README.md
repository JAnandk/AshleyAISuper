# CPG360 Advisor Dashboard

A visually immersive and interactive analytics dashboard for Consumer Packaged Goods (CPG) companies to gain actionable intelligence from retail analytics, channel partner data, and consumer engagement metrics.

## Overview

CPG360 Advisor is a confidential intelligence platform designed to bridge the data gap between CPGs and retail channels, providing real-time insights through a privacy-preserving confidential computing architecture.

## Features

### 📊 Executive Dashboard
- **Real-time KPI Metrics**: Track Total Revenue, Active SKUs, Store Coverage, and Promotion Compliance
- **Performance Trends**: Visualize sales performance vs targets with interactive charts
- **Category Analytics**: Monitor sales and growth across product categories

### 🎯 Actionable Intelligence
- **Smart Alerts**: Critical, warning, info, and success alerts for immediate action
- **AI Recommendations**: Data-driven insights for assortment optimization, promotions, inventory, and placement
- **Confidence Scoring**: Each recommendation includes confidence levels to guide decision-making

### 🗺️ Regional Insights
- **Multi-Region Dashboard**: Track performance across North, South, East, and West regions
- **Store Metrics**: Monitor revenue, store count, compliance rates, and growth per region

### 📈 Consumer Engagement
- **Foot Traffic Analytics**: Track customer engagement by time of day
- **Conversion Metrics**: Monitor engagement-to-conversion ratios
- **Product Performance**: View top-performing products with stock levels and engagement scores

### 🔐 Security & Privacy
- **Confidential Computing**: Powered by privacy-preserving data integration
- **Secure Mode Indicator**: Visual confirmation of secure data processing

## Technology Stack

### Frontend
- **React 19**: Modern React framework
- **Tailwind CSS**: Custom design system with professional color palette
- **Recharts**: Interactive charting library
- **Lucide Icons**: Comprehensive icon set

### Design System
- Professional blues, greens, and teals for trust and growth
- Typography: Inter for body, Space Grotesk for headings
- Full light and dark mode support

## Dashboard Components

### KPI Cards
Display key metrics with trend indicators and color-coded changes

### Charts
- **Sales Trend**: Area chart showing revenue vs target vs last year
- **Category Performance**: Color-coded bar chart
- **Engagement Analytics**: Multi-line chart for traffic and conversion

### Data Panels
- **Top Products**: Ranked list with sales, growth, stock, and engagement
- **Regional Performance**: Grid of region cards with metrics
- **Actionable Alerts**: Prioritized alerts with action buttons
- **AI Recommendations**: Smart suggestions with confidence scores

## Responsive Design

Fully optimized for:
